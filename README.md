# 🎮 Neon Showdown: Laser Arena

A futuristic **2-player browser game** powered by **JavaScript**, **HTML5 Canvas**, and **MediaPipe Hands**. Players control their paddles using **real-time hand tracking** through a webcam and fire laser shots using open-palm gestures.

## 🚀 Features

- ✋ Real-time hand tracking using MediaPipe Hands
- 🎯 Gesture-based controls (Move & Shoot)
- 👥 Two-player gameplay on a single webcam
- ⚡ Laser shooting mechanics
- 📊 Live score tracking
- 🎨 Neon cyberpunk-themed interface
- 🔄 Reset match functionality
- 🌐 Runs directly in the browser (No installation required)

---

## 🛠️ Technologies Used

- HTML5
- CSS3

- HTML5 Canvas API

---

## 🎮 How to Play

### Player Controls

- **Move Hand Up/Down** → Move your paddle
- **Open Palm** → Fire laser

### Gameplay

- Left side of the webcam controls **Player 2 (Blue)**.
- Right side of the webcam controls **Player 1 (Red)**.
- Hit your opponent with laser shots to score points.
- The player with the highest score wins.
